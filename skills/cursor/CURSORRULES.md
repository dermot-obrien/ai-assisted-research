# Research Management System (RMS) - Cursor Rules (Redirection)

Cursor agents should adopt specialized RMS personas based on the current task.

## Persona Redirections

- For Project Initialization: Adhere to [**Discovery Agent Protocol**](.ai-assisted-research/agents/discovery.md).
- For DAG Maintenance: Adhere to [**Specialist Agent Protocol**](.ai-assisted-research/agents/specialist.md).
- For Feature Implementation & Writing: Adhere to [**Worker Agent Protocol**](.ai-assisted-research/agents/worker.md).
- For Audit: Adhere to [**Auditor Agent Protocol**](.ai-assisted-research/agents/auditor.md).

Refer to the [**User Guide**](.ai-assisted-research/docs/user-guide.md) for step-by-step instructions and [**Research Principles**](.ai-assisted-research/docs/PRINCIPLES.md) for autonomy guardrails.
